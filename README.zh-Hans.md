# Nekomakase - 支持

**Nekomakase**是一款结合2048合成玩法与寿司烹饪的手机益智游戏。滑动合成食材，发现新料理，为顾客上菜来提升声望。

## 支持

如需帮助或反馈问题，请在此仓库创建Issue：

**[提交支持请求](https://github.com/Yichen-D/Nekomakase-support/issues/new)**

报告Bug时，请提供以下信息：
- 设备型号（如iPhone 15、Pixel 8）
- 系统版本（如iOS 17.4）
- 问题描述及预期行为

## 常见问题

**Q：怎么玩？**
向任意方向滑动，移动并合成棋盘上的食材。相匹配的食材合成后会产生新料理。在顾客失去耐心之前将料理端上桌。

**Q：进度丢失了，能恢复吗？**
游戏进度保存在设备本地。卸载应用后无法恢复。建议保留应用以保存数据。

**Q：应用崩溃或无响应怎么办？**
请先关闭并重新打开应用。如问题持续，请创建Issue并说明崩溃时的操作。

**Q：游戏免费吗？**
是的，Nekomakase完全免费。

**Q：需要联网吗？**
不需要，Nekomakase支持完全离线游玩。
