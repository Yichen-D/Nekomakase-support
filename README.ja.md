# Nekomakase - サポート

**Nekomakase**は、2048パズルと寿司づくりを組み合わせたモバイルゲームです。スワイプで食材を合成し、新しい料理を発見して、お客さんに提供しましょう。

## サポート

お困りの点や不具合の報告は、以下からIssueを作成してください。

**[サポートリクエストを送る](https://github.com/Yichen-D/Nekomakase-support/issues/new)**

不具合を報告する際は、以下の情報をお伝えください：
- 端末の機種（例：iPhone 15、Pixel 8）
- OSのバージョン（例：iOS 17.4）
- 発生した状況と期待される動作の説明

## よくある質問

**Q：遊び方を教えてください**
任意の方向にスワイプして、ボード上の食材を移動・合成します。対応する食材同士が合成されると新しい料理ができます。お客さんの忍耐が切れる前に料理を提供しましょう。

**Q：進行データが消えました。復元できますか？**
ゲームの進行データは端末にローカル保存されています。アプリを削除すると復元できません。データを保持するため、アプリをインストールしたままにすることをおすすめします。

**Q：アプリがクラッシュ、または応答しません**
アプリを一度終了して再度開いてください。改善しない場合は、クラッシュ時の操作内容を添えてIssueを作成してください。

**Q：無料ですか？**
はい、Nekomakaseは無料でダウンロード・プレイできます。

**Q：インターネット接続は必要ですか？**
いいえ、完全オフラインで遊べます。
