# Nekomakase - Support

[日本語](README.ja.md) | [简体中文](README.zh-Hans.md)

**Nekomakase** is a mobile puzzle game that combines 2048-style merge mechanics with sushi cooking. Swipe to merge ingredients on a grid, discover new dishes, and serve customers to build your reputation.

## Support

If you need help or want to report an issue, please open an issue on this repository:

**[Open a Support Request](../../issues/new)**

When reporting a bug, please include:

- Device model (e.g., iPhone 15, Pixel 8)
- OS version (e.g., iOS 17.4)
- A description of what happened and what you expected

## FAQ

**Q: How do I play?**

Swipe in any direction to move and merge ingredients on the board. When compatible ingredients merge, they create new dishes. Serve completed dishes to customers before their patience runs out.

**Q: My progress was lost. Can I recover it?**

Game progress is stored locally on your device. If you uninstall the app, your progress cannot be recovered. We recommend keeping the app installed to preserve your data.

**Q: The app crashed or is not responding. What should I do?**

Try closing and reopening the app. If the problem persists, please open an issue with details about what you were doing when the crash occurred.

**Q: Is this game free?**

Yes, Nekomakase is free to download and play.

**Q: Is an internet connection required?**

No, Nekomakase works fully offline.
